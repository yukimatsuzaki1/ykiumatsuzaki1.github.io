    オブジェクト

    どのような種類の値でも格納できるもの

    具体的には、数値・文字列・ブーリアン（真偽値）など

    const info = { name: "松崎 祐樹", age: "32" };

    オブジェクトは{}中括弧で周りを囲む

    中身は、（キー：値） ペアとなっていて、,で区切り複数の情報を格納できる

    オブジェクトの値へのアクセスの仕方(取出し方)

    ブラケット記法

    info[name] と記述すると "松崎 祐樹" が取り出せる

    info[age] と記述すると "32" が取り出せる
